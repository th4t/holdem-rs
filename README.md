#holdem-rs

A library for Rust containing Texas Hold'em specific types and logic (maybe in the distant future).

The crate is called `holdem` and you can depend on it via cargo:

```ini
[dependencies.holdem]
git = "https://github.com/th4t/holdem-rs.git"
```
